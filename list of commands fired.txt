Here's how to fix the suspend issue:
Option 1: Temporary test (requires sudo)
echo s2idle | sudo tee /sys/power/mem_sleep
systemctl suspend
Option 2: Permanent fix (recommended)
1. Backup GRUB config:
sudo cp /etc/default/grub /etc/default/grub.backup
2. Edit GRUB config and add mem_sleep_default=s2idle to GRUB_CMDLINE_LINUX_DEFAULT:
sudo nano /etc/default/grub
   Find: GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
   Change to: GRUB_CMDLINE_LINUX_DEFAULT="quiet splash mem_sleep_default=s2idle"
3. Update GRUB:
sudo update-grub
4. Reboot and test suspend.
This fixes the issue because s2idle mode works better with your ASUS laptop's dual graphics setup, while S3 (deep) mode has compatibility problems.
